1. git clone releases from github
2. sh install_all.sh --> installs django and everything
3. create a settings_<site>.py file for the site you are going to use
4. add to settings_<site>.py database, admins, debug, time_zone, language_code,
languages, MEDIA_ROOT, EMAIL_BACKEND and other email settings
5. copy gntimages (base images) to MEDIA_ROOT
6. set SPATIAL_REFERENCE_SYSTEM_ID to the right value (e.g. 3067)
7. python manage.py syncdb --settings="geonition.settings_<site>"
8. python manage.py collectstatic --settings="geonition.settings_<site>"
9. python manage.py test --settings="geonition.settings_<site>" #should pass all the tests


To migrate data from previous version dump the data into the data folder
with the names mentioned below. Load the files in the following way:
1. python manage.py dumpdata sites --settings="geonition.***" > ../data/sites.json
2. python manage.py dumpdata dashboard --settings="geonition.***" > ../data/dashboard.json
3. python manage.py dumpdata base_page --settings="geonition.***" > ../data/base_page.json
4. python manage.py dumpdata plan_proposals --settings="geonition.***" > ../data/plan_proposals.json
5. python manage.py dumpdata group --settings="geonition.***" > ../data/group.json
6. python manage.py dumpdata users --settings="geonition.***" > ../data/users.json
7. python manage.py dumpdata geonition_utils --settings="geonition.***" > ../data/geonition_utils.json
8. python manage.py dumpdata geojson_rest --settings="geonition.***" > ../data/geojson_rest.json
9. python manage.py dumpdata maps --settings="geonition.***" > ../data/maps.json
10. python manage.py dumpdata geoforms --settings="geonition.***" > ../data/geoforms.json

The final command will save the data to the new database
11. python manage.py data_migrate '/absolute/path/to/data/folder/' --settings="geonition.***"
